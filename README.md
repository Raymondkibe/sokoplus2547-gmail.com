## Quick Deployment to Vercel

1. `package.json` - Dependencies
2. `index.js` - Main backend code

4. `vercel.json` - Vercel configuration

### Step 2: Deploy to Vercel
```bash

npm i -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod
